# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
* リポジトリは保存場所
* リモートリポジトリはサーバ上にあり共有ができる
* ローカルリポジトリは各々のPC

## プッシュとマージの違いは何でしょうか？
* プッシュはローカルからリモートへ変更内容を反映
* マージはリモート同士またはローカル同士で変更内容を反映
## コミットとプッシュの違い
* 自分のブランチの変更点を保存がコミット
* ローカルからリモートへ変更がプッシュ


## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
* 変更点
* なぜ変更をしたのか
* 変更する前と後でどういうことになるのか
* 上記が、第三者に分かるように

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
* 重要度、手間数、リスクを考える
* 共有しているリポジトリを変更するにはリスクがある



## コンフリクトを起こしてしまった場合、どう対処すべきですか？
* 先にマージされた変更内容を取り込む
* 後にマージしようとしている変更内容を取り込む
* どちらの変更内容も取り込む
