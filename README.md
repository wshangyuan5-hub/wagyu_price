# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
* リポジトリは保存場所
* リモートリポジトリはサーバ上にあり共有ができる
* ローカルリポジトリは各々のPC

## プッシュとマージの違いは何でしょうか？
* データ変更前か後か
* 第三者が変更点を確認でき事前にバグを防ぐ事ができる


## コミットとプッシュの違い
* add commit push の順番で実行
* セーブするのか、上書きするのか
* 何を変更したのか一目で分かる


## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
* 変更点
* なぜ変更をしたのか
* 変更する前と後でどういうことになるのか
* 上記が、第三者に分かるように

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
* 第三者が確認してマージするかどうか



## コンフリクトを起こしてしまった場合、どう対処すべきですか？
* 先にマージされた変更内容を取り込む
* 後にマージしようとしている変更内容を取り込む
* どちらの変更内容も取り込む
